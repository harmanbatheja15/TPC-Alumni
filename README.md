# Thapar Polytechnic College - Alumni Association

This Alumni Association website is a vibrant community of former students who have graduated from Thapar Polytechnic College, Patiala. This website is designed to keep our alumni engaged and connected to each other.

On this website, alumni can find information about upcoming events, news and updates from the college, opportunities for volunteering and mentorship, and ways to give back to the institution. We also have a directory of alumni, where members can search for and connect with fellow graduates.

The website is built using modern web development technologies, including HTML, CSS, Bootstrap, JavaScript, and Django. It is responsive and optimized for a variety of devices, including desktops, tablets, and smartphones.

## Installation

To install this website, follow these steps:

1. Clone the repository: `git clone https://github.com/harmanbatheja15/TPC-Alumni`
2. `cd TPC-Alumni/alumniAssociation`
3. Create a virtual environment: `virtualenv env`
4. Activate the virtual environment: `source env/bin/activate`
5. Install the dependencies: `pip3 install django`
6. `python3 manage.py makemigrations`
7. `python3 manage.py migrate`

## Running the website

To run the website, follow these steps:

1. `cd TPC-Alumni/alumniAssociation`
2. Activate the virtual environment: `source env/bin/activate`
3. Start the development server: `python3 manage.py runserver`
4. Open your web browser and go to `http://localhost:8000/`
